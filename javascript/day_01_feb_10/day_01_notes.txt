Web Development
===============
-Developing Websites 

1. FrontEnd : User Interface ( Look & Feel )
        HTML : Content ( div, paragraph, image, table, list, form )
        CSS :  Styling ( border , background, width , height, margin, padding )
        Bootstrap : CSS Library (Twitter) Pre-defined CSS classes
        Javascript : Behaviour of a webpage
        Angular/React : Javascript Frameworks ( RAD , CLI, Form , Routing )

2. BackEnd : Develop APIs ( java, .net, python, PHP, NodeJs, Go )

3. DataBase : Holds the data ( oracle , mysql , mongoDB )
